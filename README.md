# 에너지보고대상 건물과 건축물대장 매칭 프로젝트

## 프로젝트 개요

본 프로젝트는 한국부동산원 그린리모델링 지원 사업의 일환으로, 에너지보고대상(EBD) 건물 데이터와 건축물대장(BD) 표제부 데이터를 매칭하기 위한 다단계 알고리즘을 시작으로 다양한 데이터 분석 및 시스템 구축 작업을 수행합니다. 초기 매칭 알고리즘 개발을 통해 건물 식별 체계를 확립한 후, 건물 및 에너지 데이터 EDA, 계량기 DB 시스템 구축, 그린리모델링 성과평가 시각화, 에너지 계통구조도 보완 등 종합적인 데이터 기반 접근법을 구현합니다. 이러한 다각적 작업을 통해 건물 에너지 성능 향상과 탄소중립 목표 달성을 위한 데이터 기반 의사결정 체계를 구축합니다.

### 매칭의 어려움과 중요성

에너지보고대상 건물과 건축물대장 간의 매칭 작업은 여러 도전과제를 수반합니다:

- **구조적 복잡성**: 건축물대장은 총괄표제부와 개별표제부로 구분되어 있어, 하나의 총괄표제부 내 여러 개별표제부 중 에너지보고 대상 건물이 정확히 어떤 표제부에 해당하는지 식별하기 어려움
- **공통 식별자 부재**: 두 데이터 세트 간에 명확한 매칭 키(ID)가 없어 자동화된 매칭이 어려움
- **비일관적인 데이터**: 건물명, 주소, 면적 등의 정보가 양쪽 데이터에서 다르게 표현되는 경우가 많음
- **인력・시간 소비**: 수작업으로 매칭 시 과도한 인적 자원 및 시간 소요

이러한 어려움에도 불구하고 매칭 작업은 다음과 같은 이유로 중요합니다:

- **정확한 에너지 계량기 DB 구축**: 건물별 에너지 소비량을 정확히 파악하기 위한 기초 데이터 확보
- **효율적인 그린리모델링 대상 선정**: 적절한 개선 대상 건물 식별을 위한 정확한 건물 정보 필요
- **데이터 기반 성과 평가**: 여러 모델링과 시뮬레이션을 통한 그린리모델링 효과 정량화
- **탄소중립 목표 달성**: 정확한 건물 에너지 데이터를 통한 효과적인 탄소 저감 정책 지원

### 주요 담당 업무
1. 공공 건물 에너지 최적화 계량기 DB 구축을 위한 데이터 전처리 로직 개발
2. 건물 및 에너지 데이터 분석 (EDA, 통계, 시각화 등)
3. 각종 행정 업무 자동화
4. 그린리모델링 성과평가 시각화 최적화
5. 통계, 머신러닝, 시계열 모델링 보조

- **인턴 기간**: 2025년 3월 4일 ~ 2025년 6월 23일 (4개월)

## 그린리모델링 지원 사업 목표
- 건물별 에너지 소비량 데이터의 체계적 관리 시스템 통합 DB 구축
- 그린리모델링 지원 대상 공공 기축 건물과 통합 DB간 올바른 매칭 시스템
- 그린리모델링 성과 평가를 위한 데이터 기반 분석 체계 확립
- 통계 및 데이터 기반 정량 평가 / 시계열 모델링 기반 세부 평가 / 데이터 분석 기반 현장조사 체크리스트 개발을 통한 정성 평가 / 에너지 플러스(Energy Plus) 기반 시뮬레이션 평가
- 건물 에너지 성능 향상을 위한 데이터 기반 의사결정 지원
- 탄소중립 목표 달성을 위한 건물 에너지 효율 개선 가이드라인 수립

## 주요 성과
1. 에너지소비량-건물 매칭
   - 룰 기반 매칭 (1~3차, 59.85%) → 텍스트 토큰 기반 및 오차 범위 확장 매칭 (4~9차, 15.72%) → 구조적 1:1 매칭 (10차, 5.96%)
   - 매칭 알고리즘 기반 81.53% 매칭 성공
   - GPT 기반 자동화 매칭 시스템 구축
     - 프롬프트 엔지니어링을 통한 매칭 자동화 시도
     - 인간 추론이 필요한 복잡한 케이스 처리에 활용
     - 여러 텍스트 전처리 및 모델링 task에서 LLM 활용이 우수한 성능을 보였기에 시도
     - 아쉽게도 정형 데이터 처리에서는 규칙 기반 전처리 로직이 더 우수한 성능을 보임
   - 건물 식별 정확도 향상
   - 최종 10단계 매칭 알고리즘으로 81.53%의 매칭률 달성

2. 에너지소비량-계량기 매칭
   - 다중 계량기 건물 처리 시스템 구축
   - 월간/연간 에너지 소비량 데이터 검증 체계 확립


3. 통합 원가명세서 자동화
   - 자동화된 원가명세서 통합 시스템 구축
   - 다양한 형식의 원가 데이터 통합 처리 로직 개발

4. 건축HUB API 매칭
   - 건축HUB Open API를 활용한 미매칭 건물 수동 매칭 시스템 구축
   - 주소 기반 지오코딩을 통한 위치 데이터 시각화
   - API 호출 자동화를 통한 수동 매칭 작업 효율화
   - 나머지 미매칭 건물(775건)에 대한 매칭 성공률 향상
   - openAPI 활용 자동화 수집 도구 개발

5. 계량기 인프라 구축 방안
   - 계량기 시스템 계통구조도 시각화 개선
   - 문서 가독성 향상 및 오류 식별/수정 작업 수행
   - 에너지 계통 인프라 문서화 표준 수립
   - 업무 효율성 증대를 위한 시각적 문서 템플릿 개발

6. 데이터 분석 및 시각화
   - 건물 에너지 소비 패턴 분석 및 인사이트 도출
   - 그린리모델링 성과 평가를 위한 대시보드 개발
   - 에너지 절감량 예측 모델 개발 지원
   - 시계열 데이터 분석을 통한 계절적 패턴 식별
   - 건물 유형별 에너지 효율성 비교 분석
