# 건축HUB API 매칭 프로젝트

## 프로젝트 개요

본 프로젝트는 국토교통부에서 제공하는 건축HUB 건축물대장정보 서비스 API를 활용하여 그린리모델링 대상 건축물 4,194건의 정보를 수집하고 정확하게 매칭하는 작업입니다. 건축물 정보의 완전한 매칭을 통해 그린리모델링 사업 대상 건축물의 정확한 분석과 효율적인 사업 추진을 지원합니다.

## 데이터 소스

- **주요 데이터**: 국토교통부_건축HUB_건축물대장정보 서비스 ([공공데이터포털](https://www.data.go.kr/data/15134735/openapi.do))
- **API 유형**: REST
- **데이터 포맷**: JSON/XML
- **제공 정보**: 총괄표제부, 표제부, 층별개요, 부속지번, 전유공용면적, 오수정화시설, 주택가격, 전유부, 지역지구구역 등 건축물 속성정보

## 매칭 프로세스

1. **데이터 수집**: 건축HUB API를 통해 건축물대장 정보 수집
2. **데이터 전처리**: 수집된 데이터의 정제 및 표준화
3. **매칭 알고리즘 적용**: 그린리모델링 대상 4,194건과 건축물대장 정보 매칭
4. **검증 및 보완**: 매칭 결과 검증 및 정확도 향상
5. **결과 데이터 구조화**: 분석 및 활용을 위한 데이터 구조화

## 주요 기능

- 건축물대장 정보 API 연동 및 데이터 수집
- 건축물 식별자 기반 매칭 로직 구현
- 매칭 정확도 검증 및 개선
- 매칭 결과 데이터 시각화
- 매칭 완료된 건축물 정보 분석 리포트 생성

## 활용 방안

- 그린리모델링 대상 건축물의 상세 특성 분석
- 에너지 효율 개선 잠재력 평가
- 리모델링 우선순위 설정 및 전략 수립
- 사업 효과 예측 및 성과 측정 기반 마련

## 기술 스택

- 데이터 수집 및 처리: Python
- API 연동: REST API 호출 라이브러리
